Detailed Description

Wrapper around Adafruit_PN532 providing extended utility functions for NFC card operations.

This class inherits all constructors from Adafruit_PN532, allowing initialization via I2C, SPI, Software SPI, or UART. It adds convenience methods for reading UID, retrieving firmware version, and sending APDU commands to ISO14443-4 cards.

Retrieve the firmware version of the PN532 module.

The firmware version is returned as a 32-bit value where:

  • Bits 31:24 = IC type
  • Bits 23:16 = Major firmware version
  • Bits 15:8 = Minor firmware version
  • Bits 7:0 = Supported feature flags
