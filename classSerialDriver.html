Detailed Description

Abstract interface for serial communication.

This class provides a hardware-agnostic interface for serial output, allowing different implementations (Arduino Serial, LCD, network logging, etc.) to be used interchangeably by higher-level code like CryptnoxWallet.

Implementations must provide all print/println variants used for debug output.
